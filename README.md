# FunctionalPrograming

函数式编程学习笔记及Demo
主要笔记存在于
- 函数式编程指北 —— 介绍什么是函数式编程以及为什么使用函数式编程
- ramda.js —— 挑选 ramda.js 中应用概率较高的部分

## 学习路线

1. 《函数式编程指北》了解函数式编程思想
2. ramda.js 学习函数式编程相关的工具的使用
3. 借助函数式编程完成一个实践。将实际业务中的一个模块改造为函数式编程范式
